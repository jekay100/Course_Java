static 修饰词（关键字）是用业修饰类的成员的
静态修饰的成员变量，是属于类级别的变量，，是在类的加载期间
分配的，
没有static的里面的属性是属于某个实例的因为每个方形的长宽高是不一样的
不是属于类的，每个对象全不同，所以就实例变量，属于对象的属性

静态代码块，在类体内定义在类加载时运行只运行一次

final最终的，电视的大结局叫final
final 可以修饰类，方法变量，static 变量方法，代码块，内部类
final修饰的类不能被继承，（最终的，剧终了所以不能继承了）
String Math这两个类全是final不能被继承了
final类不建议使用，
final方法不能被覆盖了，因为剧终了
final变量初始化以后不永许修改了，只能读
final int a = 1;a++就不对了
